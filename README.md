# Mermaid ASCII Service

A Vue 3 web application for rendering Mermaid diagrams in ASCII format. This service provides an interface to create, edit, and visualize Mermaid diagrams using the [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) library.

## Running the Application

To build and run the application in Docker:

```bash
./run.sh
```

This will:
```
- Build the Docker image
- Start the container on port 9001
- Make the application available at http://localhost:9001

Press Ctrl+C to stop the container.
```

Enjoy! :blush:
